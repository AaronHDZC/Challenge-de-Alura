# 📊 Análisis de Optimización de Negocio: Toma de Decisiones para el Sr. Juan

Este proyecto presenta un análisis de datos exhaustivo realizado para el **Sr. Juan**, propietario de una red de tiendas comerciales realizado como arte del Challenge Alura Store de la Plataforma Alura LATAM. El objetivo primordial es evaluar el desempeño operativo de sus cuatro sucursales para identificar cuál de ellas presenta el menor rendimiento y recomendar su cierre o venta estratégica basándose en evidencia cuantitativa.


## Propósito del Proyecto
El Sr. Juan requiere optimizar sus inversiones. Para ello, se procesaron cuatro bases de datos independientes para comparar factores críticos que afectan directamente la rentabilidad, la logística y la experiencia del cliente en cada una de sus ubicaciones.

## Estructura del Análisis
El flujo de trabajo en Python y Pandas se divide en las siguientes etapas:

1.  **Carga de Datos:** Importación de registros de ventas por sucursal.
2.  **Análisis de Facturación:** Comparativa de ingresos brutos totales.
3.  **Análisis de Inventario:** Determinación de las categorías de productos con mayor y menor movimiento.
4.  **Satisfacción del Cliente:** Cálculo de la calificación promedio por tienda para medir la lealtad y el servicio.
5.  **Desempeño de Productos:** Identificación de productos "estrella" y artículos de baja rotación.
6.  **Auditoría Logística:** Evaluación del costo de envío promedio por tienda (costo asumido por el cliente).
7.  **Dictamen Final:** Informe ejecutivo con la recomendación de cierre sustentada.

## Herramientas Utilizadas
* **Python 3**
* **Pandas:** Procesamiento y limpieza de datos.
* **Matplotlib:** Visualización de tendencias y comparativas.
* **Google Colab:** Entorno de ejecución y análisis.

## Hallazgos y Recomendación
Tras el análisis técnico, se recomienda al Sr. Juan el cierre de la **Tienda 4**. Los datos demuestran que esta sucursal posee:
* La facturación más baja de la red.
* Un índice de satisfacción del cliente crítico (calificaciones más bajas).
* Costos de envío significativamente superiores a las otras tiendas, lo que desincentiva la compra y afecta la competitividad.
